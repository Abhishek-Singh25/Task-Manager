Task Manager App
A full-stack Task Manager application built using the MERN stack (MongoDB, Express.js, React, and Node.js).
The application uses JWT Authentication, allowing users to securely register and log in. Each user can access and manage only their own tasks.
Users can create, update, delete, and track tasks by changing their status between Pending and Completed through a clean user interface.

Tech Stack
Frontend: React, Axios, React Router Dom, CSS
Backend: Node.js, Express.js, JWT, cors, bcryptjs, dotenv
Database: MongoDB Atlas, Mongoose
